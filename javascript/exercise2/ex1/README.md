- Here is a sample html file with a submit button. Now modify the style of the paragraph text through javascript code.
```

<html>
  <head>

    <title>JS DOM paragraph style</title>
  </head> 
  <body>
    <p id ='text'>JavaScript Exercises - w3resource</p>

    </div>
  </body>
</html>
```
